## 🚀 见证软件演进的力量：北京大学软件复用小组主页 v1.0 正式发布！
经过团队成员的共同努力，北京大学软件复用小组（PKU Software Reuse Group） 官方主页 v1.0 版本现已正式上线。

作为我们在软件工程学术研究与工程实践道路上的新起点，该主页不仅是小组科研成果的展示窗口，更是我们与国内外学术界、工业界连接的桥梁。

## 🌐 访问地址
[https://PKU-SoftwareReuse.github.io]

## ✨ 亮点特性
1. 核心科研方向全景展示

    - 软件复用与特征化演进 (Software Reuse & FOSD)

    - 大语言模型辅助软件工程 (LLM4SE)

    - 代码分析与图表示学习 (GNN for Code Analysis)

    - 智能编程辅助工具 (FeatX, CodeAgent)

2. 学术成果归档
收录了小组历年来在 ICSE, FSE, ASE, ICSME 等顶级学术会议与期刊发表的重要论文，并提供部分项目的开源代码入口。

3. 现代化响应式设计
采用简约而不简单的学术风 UI 界面，完美适配 PC、平板与手机移动端，随时随地查阅最新科研进展。

## 🛠️ 技术底座
本站 v1.0 版本基于以下技术栈构建：

框架： Jekyll

部署： GitHub Pages

样式： 简洁的 Markdown 驱动内容管理

## 📢 结语
主页的上线只是一个开始。未来，我们将继续探索软件工程的边界，致力于通过 特征驱动 (Feature-oriented) 的视角，让复杂软件系统的维护与演进变得更加智能与高效。

欢迎各位师生、同行点击访问并提出宝贵建议！
